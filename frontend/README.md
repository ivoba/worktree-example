# Worktree Manager Frontend

TypeScript + Vite frontend for the git worktree manager example.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Copy `.env.example` to `.env` and adjust `VITE_API_URL` if needed.

By default, the Vite dev server proxies `/api` and `/health` to the PHP API.
